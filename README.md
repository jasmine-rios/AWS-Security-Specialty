# AWS Certified Security - Specialty (SCS-C03)

Study material for AWS Security Specialty for renewal for 2026.

Remember to use the 50% off voucher when scheduling the certification as the test is $300.
Use your work email to see if you have a discount as well.

The types of questions on the exam are:
- Multiple choice
- Multiple Response
- Ordering
- Matching

There is now a digital notes tool on the exam that allows you to organize your thoughts or make general notes about a question.

For renewal, take the practice exam first to see what you need to focus on studying.

My score at the begining of testing: 65%
